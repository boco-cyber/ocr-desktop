Ollama
Run open-source LLMs locally for OCR and translation. No API key needed. Ollama must be running on your machine.
Default: http://localhost:11434. Change if Ollama runs on a different port or remote host.
Click "Test Connection" above to discover available models.
Vision-capable models (marked vision) are required for OCR. Text-only models work for translation.
Installation
winget install Ollama.Ollama
ollama pull llava  # vision model for OCR
ollama pull llama3.1  # text model for translation
Download Ollama  ·  Browse Models
PaddleOCR
High-accuracy local OCR via PaddleOCR (Python). Requires Python with paddlepaddle and paddleocr installed.

Must have pip install paddlepaddle paddleocr installed. Leave blank to use the Python path from General.

Click "Test Python" to verify your PaddleOCR environment.
Tesseract OCR
Pure local OCR — no Python, no API key, no installation required. Runs directly inside the app using Tesseract.js.
No setup needed

Tesseract.js runs entirely inside Node.js — no external Python, no subprocess, no DLL conflicts.
Language data is downloaded automatically on first use and cached locally. Supports 100+ languages.

Always available. Select Tesseract (Local / Free) as the OCR provider to use it.
NLLB Translation
Facebook's No Language Left Behind model — runs fully offline. Requires Python with transformers and torch.

Leave blank to use the system Python. Must have pip install transformers torch sentencepiece installed.

Set a limit if you run out of VRAM. 0 = no limit.

600M — Fast and lightweight, good for most use cases.
1.3B — Better translation quality, needs ~3 GB RAM.
3.3B — Best quality, needs ~8 GB RAM / VRAM.
Click "Test Python" to check your environment.
